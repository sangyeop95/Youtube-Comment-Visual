# Youtube-Comment-Visual
유튜브 영상이나 shorts에 달린 댓글내용중에서 자주 언급된 키워드를 빠르게 확인해 볼 수 있습니다

작업자 : 차상엽(1인)

1. 유튜브 영상 댓글 시각화
2. 좋아요 댓글차트
   
좀 더 정확한 댓글 키워드 분석을 위해서는 KONLPY의 OKT 형태소 분석 사용을 위해 *JDK설치* 이후 시스템 변수 설정 및 재부팅, 재실행을 해주세요.
